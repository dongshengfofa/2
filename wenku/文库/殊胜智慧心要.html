殊胜智慧心要

作者：佚名　来源：佛学在线


　　
　　
　　***************智慧心要**************

　　大圆满 教法最初是在清净的色究竟天密严净土中，由报身佛普贤如来通过直指心性之密意传承( 心传心 ) 传授予诸佛，以及男女菩萨。

　　在娑婆世界，大圆满教法最初的弘扬者是化身嘎绕多杰 ( 极喜金刚 )。极喜金刚出生于公元536年，而以锡兰南传佛教的记载佛陀涅磐日则是在公元 543年。 金刚手或金刚萨缍 ( 亦有续部如是记载 )， 於一时将大圆满一切续部、口传以窍诀指示毫无保留的传授给了极喜金刚。 之后，极喜金刚与空行母在乌金净土西北隅，整理了所有已挖出以及还没有现出的大圆满教法。 他们把两万部的大圆满续部分类成六万四千个偈颂。

　　极喜金刚的弟子蒋巴舍宁 ( 文殊友 )， 再把六万四千偈颂分成了三大部：A) 外心识部 ，B) 内法界部，C) 极密窍诀部。

　　A) 外心识部：虽然心部可以分类成二十一部，可是当心法在西藏翻译之时， 它被分类成十八部的母续与子续。 而十八部母续与子续的前五续是由Vairochana 所翻译并立名为“前译五续”。 加上其弟子Yudra Nyingpo所翻译的“后译十三续”，和三部主要的续部，就成为了二十一部续部。

　　Vairochana 是在上师夕阿星哈那儿得到心识部的教法，再传给了藏王赤松德赞，Yudua Nyingpo与其他人。之后，此教法传遍了整个西藏。

　　B) 内法界部：内法界部共有两万卷，分成：白法界，黑法界 ， 杂法界。九个法界。

　　Vairochana 在夕阿星哈那儿得到了口传，而写了“金刚桥”( The Vajra Bridge ) 这部经典。 之后把这部经典传给了西藏的 Pang Sangye Gongpo 。Pang Sangye Gongpo 与他的六个传承弟子皆都成就了虹光身。

　　C) 极密窍诀部：最后蒋巴舍宁 ( 文殊友 ) 的弟子西日桑哈将它分成：
　　1) 外续 ，就如同外形的身体
　　2) 内续，就如同眼睛
　　3) 秘密续，就如同心
　　4) 集密窍诀续，就如同包容一切的完整身体。

　　这四部与 Trekcho 教法一般是相同，而仍有的不同点是在明晰(clarity)，明确explicitness，与Togal 部分的详细教法。

　　夕阿星哈将内、外与秘密续同时传给了迦纳苏扎，和毗玛拉迷。於此同时还把极密窍诀传给了迦纳苏扎；后者再传给了毗玛拉迷。

　　*************毗玛心要****************

　　此传承的首四位导师在离开这娑婆世界时，都出现了很多特殊之瑞相。人间第一位导师嘎绕多杰( 极喜金刚 ) 以一道彩虹光离开，留下最后的一份教典是：《椎击三要》 Tsik Sum Ne' Dek 给文殊友 。 而当文殊友离开时，化成了一团虹光，留下最后教言给他的弟子夕阿星哈，名为：《禅定六觉》 Gom Nyam Drukpa。夕阿星哈离开时， 身体化光留下的遗教是：《七根钉子》 Zerbu Pun” 给迦纳苏扎。迦纳苏扎离开时，化成一团光留下的最后教言是：《建立四方法》Shyak Tab Shyipa。

　　从本初佛普贤如来传给金刚手或金刚萨缍到极喜金刚的传承，名为直指心意的胜者密意传承 – Gyalwa Gong Gyu。 从极喜金刚传给文殊友， 夕阿星哈，到毗玛拉迷，这样的传承名为持明指示传承 -- Rigdzin Da Gyu。毗玛拉迷之后，此传承则名为补特伽罗耳传承 -- Gangzak Nyen Gyu。这三个是属于毗玛法脉的传承。

　　极内密窍诀续可分成十七部大续。毗玛传承加入了《忿怒母续》， 因而成为了十八部大续。莲师传承则把《法界光明炽燃续》加入了这十七部之中。 后来，毗玛拉迷的《忿怒母续》和莲花生大士的《法界光明炽燃续》加上十七部大续，就成为了十九部。

　　在西藏大成就者如大班智达毗玛拉迷(无垢友)和莲花生大士，把这“心要光明大圆满”传遍了西藏。

　　在桑耶寺的Utse' Barkhang 的房间内，毗玛拉迷 ( 无垢友 ) 把“大圆满极密心要窍诀”的甚深教义秘密的传授给了：国王赤松德赞，太子牟尼赞波，杨.括晋桑波，卡瓦贝择，措罗鲁嘉昇等五位具缘者。毗玛传承是从这“心要”法脉传下来。包括了：
　　1) 金字续 ( golden letters instructions )
　　2) 松耳石续 ( turquoise letters instructions )
　　3) 铜字续 ( copper letters instructions )
　　4) 海螺字续 ( conch letters instructions )

　　这亦名为“四大甚深心要”。( 后来成为了极密窍诀续之一百十九种修法 )

　　毗玛拉迷 ( 无垢友 ) 在西藏住了十三年，并承诺每三百年化身来教导遗留在五台山的光明大圆满教法。因此，他以虹化身住世直到这劫( Fortunate kalpa )1000 佛全部出世。当这一切完成后，他将再回到印度的 Vajrasana 完成最后的证悟之道。

　　毗玛拉迷 ( 无垢友 ) 赴五台山五十五年后，传承从 ( Nyangben Tengdzin Zangpo ) 扬本恬晋桑波传给了贝罗多旺秋。随后他先把耳传的所有续部都伏藏起来，接着是以虹化身离开。而贝罗多旺秋把极密窍诀传给了当玛沧珠加森。当玛沧珠加森传给了至尊桑给旺波。至尊桑给旺波传给了彰东扎西楚巴 ，后便化虹身而去。Gyalwa Shyangton Tashi Dorje 大成就者 ( 彰东扎西楚巴 ) 又传给了给巴义本。 之后，又传给了古汝著巴。古汝著巴又传给了大成就者楚西桑给严巴。楚西桑给严巴又传给了弥隆多杰。弥隆多杰再传给了格玛燃扎尊者 。 后来，格玛燃扎尊者再传给全知无垢友尊者。这就是此法脉传承到公元 1308 年全知无垢友出生那年的情形。

　　一般上在“新”与“旧”心要中，毗玛心要是属于旧的心要；同时于耳传和伏藏之分别中，它是属于伏藏的部分。它也属于母续的教法。

　　“毗玛心要”亦名为“大圆满珍珠鬘”( Garland of Peals' )。龙钦巴尊者根据“大圆满珍珠鬘”所作的注解称为“甚深上师心要”，又称为“如意宝”( Lama Yangtig - Wish Fulfilling Jewel' )，并可分成五十一种修法。由于它是源于“毗玛心要”，因此它是属于“子续”的教法。


　　*************空行心要************

　　夕阿星哈 将此“心要”的传承授予莲花生大士 。莲花生大士在哟多特卓山洞中将整部“光明大圆满心要”秘密的传给了空行“益西措嘉”和十万空性母与护法众。

　　当时，赤松德赞之女“贝玛萨”突然死了，死时才八岁。国王非常悲伤，就去祈求莲花生大士帮助。莲花生大士在公主的心间画了个“尼”(NRI)字，将公主的神识以禅定勾召回来。当公主恢复呼吸及能开口说话时，莲花生大士将整部“空性心要”传授予公主，还直接赐予加持，并给公主取了密名贝玛罗德萨。得到圆满传承的公主于是从师而发愿，于后世将把这伏藏法挖掘出来利益无量众生，就圆寂了。之后，莲花生大士把这“空性心要”全部埋藏起来成为伏藏。

　　公主后来转世成为大伏藏师贝玛罗德萨。他将此空行心要于一个叫波及阐达哈答目达的地方挖掘了出来。之后传给了大弟子嘉昇仁真(Gyalse Lekden)。贝玛罗德萨又转世为全知无垢光尊者(Longchenpa)。他从嘉昇仁真那儿得到了整部“空行心要”的教法。因此，这传承能够继续流传下去。

　　根据空行心要，佛以心性传达之意义与密严刹土的大圆满教义是相同的。持明表示传承是从金刚手给极喜金刚，到夕阿星哈，到莲花生大士，后来传给了益西措嘉和贝玛萨公主。从她又传给了贝玛罗德萨，嘉昇仁真，和无垢光尊者。

　　“空行心要”有六十五个分类。它是由十二部母续与子续“Takdrol Gyu”，以及佛所说的“遗教三经”和其他教法结合而形成的。 极密窍诀藏续是以这十二部的母续与子续“Takdrol Gyu”和“遗教三经”做为空行心要的参考资料。

　　莲花生大士的空行心要是名为“新”心要也是属于伏藏品。 当传给益西措嘉和贝玛萨的当时，她们都是空性母和护法神(Shaza Khamoche)，于是这教法就称为空行心要( Khandro)。此莲花生大士的空行心要也称为母续，和毗玛心要一起称为二部母续。

　　这两部母续和子续合起来被称为“四部心要”，后来称为“心要”。之后，无垢光尊者写了部“Zambo Yangtik”，内容虽然包括了最重要的两部空行和毗玛，然而却没有做详细的说明。

　　多数的内部秘密心要都属于伏藏教法，可分为短，中，和广的教法。最广者是智慧心要Nyingthig Ysbshyi，中广者是南方之宝Gongpa Zangtal，“ All Penetrating Wisdom Mind”和浓缩者是为阿底实相之精华(Essence Of Profound Truth Of Ati)。

　　**********教法的传承历程**********


　　传承从龙钦巴尊者到吉美林巴，总共有十四位传承主师。吉美林巴在观修无上龙钦巴的三年期间，龙钦巴尊者化现在吉美林巴前三次，还将整部心要传授给他。这直接从龙钦巴尊者传给吉美林巴的传承是属于“短”传承。

　　从吉美林巴尊者传给Jinkme Gyalwe Nyugu，之后传给Jikme Chokyi Wango ( Patral Rinpoche)。Patral Rinpoche再传给了Khenchen Ngekai Wangpo( Khenpo Ngaga )。Khenpo Ngaga 再传给弟子 Shedrup Lungtok Tenpe Nyima，他也是 Nyoshul Lungtok 的转世。他把大部分的心要教义传授予贝诺法王 HH Penor Rinpoche。

　　Nyoshul Lungtok Tenpe Nyima 传给 Khenchen Ngaga，Khenchen Ngaga 接下来传给 Palchen Dupa 第二世的 Pema Norbu Ricpoche。

　　Khenpo Ngaga 与 Palchen dupa 二位同是老师与弟子的关系。所以，Palchen Dupa 在 Khenchen Ngaga 那儿得到很多领体的教言； 同样的，Palchen Dupa 也把教言传授给 Khenchen Ngaga。 现在 Palchen Dupa 传授给 Thubten Chokyi Dawa 第二世的 Choktrul Rinpoche。Choktrul Rinpoche再传给来了贝诺法王HH Drubwang Penor Rinpoche。

　　因此，这两个从 Khenpo Nagag 之传承就变成了一个。最后，贝诺法王接受了完整的智慧心要的教言。同时，从顶果钦哲仁波切 Dilgo Khyentse Rinpoche 那儿得到了详细的法本。
